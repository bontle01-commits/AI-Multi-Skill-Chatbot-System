# AI-Multi-Skill-Chatbot-System
AI Multi-Skill Chatbot System

A modular AI chatbot built with FastAPI, Machine Learning, and PostgreSQL that detects user intent and routes conversations across multiple domains.

Features
 Intent detection using TF-IDF + Logistic Regression
 Travel assistant with step-by-step booking flow
 Tech support, university FAQ, mental health support
 Context-aware (multi-turn) conversations
 PostgreSQL for chat storage and analytics
 FastAPI backend with REST endpoints

How It Works
User - ML Model - Intent - Router - Skill - Response - Database

Tech Stack

Python, FastAPI, Scikit-learn, PostgreSQL, SQLAlchemy
